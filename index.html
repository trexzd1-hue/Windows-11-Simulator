Welcome to Windows 11 Simulator

This project is a fully interactive Windows 11 simulation built using pure HTML, CSS, and JavaScript. It recreates the modern Windows experience with smooth animations, draggable windows, a functional Start Menu, and a dynamic taskbar.

The goal of this simulator is to provide a realistic and responsive environment that feels close to the real Windows 11 interface while remaining lightweight and accessible directly from your browser.

More features will be added over time, including improved window management, a more advanced file explorer, and additional system apps to enhance the overall experience.

Thank you for trying this project. Enjoy exploring the simulation!
